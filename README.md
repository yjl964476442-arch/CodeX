## 模块目录

| 模块 | 用途 | 入口 |
| --- | --- | --- |
| 修复CodeX模块异常 | 修复 Windows 环境下 sandbox、代理环境变量和 `apply_patch` 相关异常 | [查看模块](./修复CodeX模块异常/README.md) |
| CodeX查询重置到期时间 | 查询 ChatGPT/Codex reset credits 的到期时间 | [查看模块](./CodeX查询重置到期时间/README.md) |
